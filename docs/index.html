Frame forwarding

by ff.kernellab.site


Forward your website using only DNS.

This can be found on github

Configure your DNS Zone as follows:

Host Record: www.example.com
Type: CNAME
To: www.new-domain.com.ff.kernellab.site

For the slash: www.new-domain.com to www.example.com/blogs

Host Record: www.example.com
Type: CNAME
To: www.new-domain.com.slash.blogs.ff.kernellab.site
